# Job Portal with Search & Filters

## 📌 Project Overview
This project is a Job Portal website that allows users to search and filter job listings easily.

## ✨ Features
- Job Search
- Category Filters
- Location Filters
- User Registration & Login
- Apply for Jobs
- Responsive Design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 Installation
1. Download the project files.
2. Open the project folder.
3. Run `index.html` in your browser.
